# CoreWeave Contracted GW Valuation Model

Live dashboard:
https://1bsar.github.io/CoreWeave-Contracted-GW-Valuation-Model/

Interactive valuation model estimating CoreWeave's intrinsic value using two independent valuation frameworks.

## Valuation Methods

### Contracted GW Method

Values the business based on contracted GPU capacity, enterprise value per contracted gigawatt, and balance sheet adjustments.

### Backlog DCF Method

Estimates intrinsic value by modeling contracted backlog conversion, EBITDA margins, buildout capital expenditures, valuation multiples, and net debt.

The model is designed as a research framework rather than a short-term price prediction.

Not financial advice.
